Apple Pay Auto-Log

Automatically log Apple Pay transactions via iOS Shortcuts.

Apple Pay Setup

1. Open Shortcuts on your iPhone

Go to Automation tab → + → Transaction.

2. Add these actions in order:

  1. Format Date — Current Date as custom: yyyyMMddHHmmss. Save to variable txId.
  2. Dictionary with 4 keys:
    • merchant → Merchant (magic variable)
    • amount → Amount (magic variable, as Number)
    • currency → Currency Code (magic variable)
    • ts → Current Date (as Unix Timestamp)
  3. Get Contents of URL:
    • URL:
    • Method: PUT
    • Request Body: JSON → the Dictionary

3. Copy your inbox URL

Paste this as the URL in the shortcut (replace {txId} with the Format Date variable):

4. Done!

Every Apple Pay transaction will appear in MaMony automatically, categorized by merchant name.
